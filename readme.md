## ✨ Overview

VisionGPT is a modern AI chatbot that combines **text conversations**, **image understanding**, and **web-powered answers** into a single beautiful interface.

Upload an image, ask questions, generate responses, browse previous chats, and interact with AI through a fast, responsive experience.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Utkarsh-webdev/VisionGPT.git

cd VisionGPT
```

---

## Install Frontend

```bash
cd client

npm install
```

---

## Install Backend

```bash
cd ../server

npm install
```

---

# 🔑 Environment Variables

Create a `.env` inside the server folder.

```env
PORT=5000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_secret

GEMINI_API_KEY=your_api_key

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```

---

# ▶️ Run

Backend

```bash
npm run server
```

Frontend

```bash
npm run dev
```
